### 개발 서버 배포를 위한 환경 변수 설정하기

> `django`를 기반으로, 개발 서버 배포를 위한 환경 변수(environment variable) 설정에 대해 설명한 문서입니다. 



#### 0. 환경 변수를 사용하는 이유

- 다른 개발자들과 원활한 협업
- 노출되어서는 안 되는 값 처리



#### 1. python-decouple

**1.1 설치**

```bash
pip install python-decouple
```



#### 2. os.environ

**2.1. .bashrc 사용**

```bash
$ vi ~/.bashrc # .bashrc 파일 수정
```



```bash
export '환경변수명 1' = '값 1'
export '환경변수명 2' = '값 2'
...
# ex> export DJANGO_SECRET_KEY = 'sgalkjlk23jlk22098(*)&&%&*'
```
